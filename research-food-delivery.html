外送系統競品分析

UX Research

深入分析 Uber Eats、foodpanda 等外送平台的核心功能、使用流程與差異化策略，提出設計改善建議。

角色：UX Researcher 類型：競品分析 / 流程拆解 方法：Task Analysis / 功能比較

研究背景與目標

  • 外送平台已成為台灣消費者日常生活的一部分，市場競爭激烈
  • 透過拆解 Uber Eats 與 foodpanda 的核心流程，分析兩大平台在使用者體驗上的差異
  • 針對搜尋、下單、追蹤、評價等關鍵流程提出設計改善建議

    • 研究方法

    方法說明
    競品選定Uber Eats、foodpanda
    Task Analysis拆解從搜尋餐廳到完成訂單的完整流程步驟
    功能比較比較搜尋篩選、餐廳資訊、訂單追蹤、優惠機制等功能差異
    策略分析分析兩平台的商業模式與差異化策略

    核心流程拆解

    Uber Eats 下單流程

    開啟 APP → 瀏覽 / 搜尋餐廳 → 查看菜單 → 加入購物車 → 結帳付款 → 即時追蹤 → 收餐評價

    foodpanda 下單流程

    開啟 APP → 選擇分類 / 搜尋 → 查看餐廳與菜單 → 客製化餐點 → 確認訂單 → 追蹤外送員 → 取餐完成

    功能比較

    功能面向Uber Eatsfoodpanda
    搜尋與篩選智慧推薦 + 多維度篩選分類明確 + 排序選項
    餐廳資訊評分 + 預估時間 + 外送費評分 + 最低消費 + 優惠標示
    優惠機制Uber One 訂閱制pandapro + 折扣碼
    訂單追蹤地圖即時追蹤進度條 + 地圖追蹤
    支付方式信用卡 / Apple Pay / LINE Pay信用卡 / 貨到付款 / LINE Pay
    特色功能團體訂單、排程預約生鮮雜貨、pandamart

    關鍵發現

    • Uber Eats 的搜尋體驗更智慧 — 推薦演算法根據歷史訂單與偏好提供個人化建議
    • foodpanda 的優惠機制更直覺 — 折扣資訊在瀏覽階段就清楚標示，降低決策成本
    • 訂單追蹤是留存關鍵 — 即時地圖追蹤大幅降低等待焦慮，兩家皆做得不錯
    • foodpanda 的生態系更廣 — pandamart 拓展到生鮮雜貨，創造更多使用場景
