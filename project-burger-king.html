豪點王 Redesign

把一個「打開就想關掉」的速食 LINE LIFF 網站，改成用戶真正會用的會員工具 — 以 Before / After 對照，讓每一處改善都有跡可循。

角色：UIUX 設計師 時間：2024.8 – 2024.11 類型：LINE LIFF Redesign

基於保密協議，本頁僅展示部分畫面與流程，不包含完整介面內容。

解決的痛點

舊版問題新版設計解法
首頁資訊雜亂，Banner 佔據大量版面會員條碼 + QR Code 置於首頁核心位置
視覺風格過時（棕色暗沈）白底 + 金黃色 CTA，乾淨現代
優惠券只顯示數量，無分類5 種狀態分類 + 使用條件說明
堡幣系統入口不明顯首頁直接顯示堡幣 + 到期提醒 + 兌換按鈕
缺少載具綁定提示區分已/未綁定載具兩種狀態
導覽列圖示不直覺5 個清楚標籤，首頁居中

使用者流程

首頁核心流程

開啟網站 → Hi, 用戶名 + 會員等級 → 堡幣餘額 + 即將到期 → 堡幣紀錄 / 開始兌換 → 會員 QR Code + 載具條碼

優惠券管理流程

底部「優惠券」→ 瀏覽可用列表 → 查看條件與到期日 → 出示給店員

設計原因

設計元素舊版 → 新版原因
主色調棕色暗沈 → 白底 + 金黃色大幅提升清爽感與品牌現代感
資訊層級Banner 優先 → 功能優先按使用頻率排列，符合 F 型閱讀模式
卡片設計平面無層次 → 圓角 + 淺陰影現代 UI 風格，提升視覺層次
底部導覽4 tab → 5 tab，首頁居中符合拇指熱區，最常用功能最易觸及
會員等級僅文字 → 徽章式標籤趣味命名增加品牌記憶點

專案亮點

  • 改了什麼、為什麼改 — Before / After 逐項對照，讓設計決策的價值不言自明
  • 把主角還給用戶 — 首頁從 Banner 廣告牆翻轉為會員功能優先，資訊層級徹底重排
  • 每種情境都有對應畫面 — 優惠券 5 種生命週期狀態 + 載具綁定雙狀態，零遺漏
  • 視覺語言的世代更新 — 棕色暗沈感退場，白底金黃 CTA 登場，品牌感瞬間年輕十歲
  • 不是重新設計，是精準手術 — 基於真實產品的痛點逐一開刀，展現 Redesign 的問題拆解與判斷力
