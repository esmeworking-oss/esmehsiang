WINGSTARS 啦啦隊官方網站

Web + Mobile

把追星變成一場收集遊戲 — 展示櫃、排行榜、投票競爭，用遊戲化機制讓粉絲從「看看就好」變成「每天都想打開」。

角色：產品設計師 時間：2024.9 – 2024.11 類型：Web + Mobile 社群平台

基於保密協議，本頁僅展示部分畫面與流程，不包含完整介面內容。

設計原因

設計元素選擇原因
主色調粉紅色 + 淺粉漸層契合女子啦啦隊品牌調性，傳達青春活力
裝飾元素星星閃爍圖示呼應「WING STARS」品牌名稱
排行榜前三名卡片式突出獎盃式排列，遊戲化視覺刺激投票
展示櫃卡片網格 + 鎖頭/CLEAR借鑒遊戲成就系統，驅動收集行為
導覽結構左側固定 + 頂部全站導覽清楚區分公共/私人功能區域

專案亮點

  • 讓人「捨不得離開」的遊戲化設計 — 排行榜競爭、展示櫃收集、CLEAR 成就解鎖，每個機制都在製造回訪理由
  • 公開與私密的精準分界 — Sitemap 清楚劃分訪客可見與會員專屬功能，權限邏輯一目瞭然
  • 桌面追榜、手機投票 — 雙端同步設計，不同場景都有最適體驗
  • 從「好想要」到「集滿了！」 — 鎖頭激發渴望、收集帶來滿足、CLEAR 給予成就感，完整情感弧線
  • 買了還想買的粉絲經濟飛輪 — 購買周邊 → 展示櫃炫耀 → 排行榜競爭 → 再次購買，商業與體驗雙贏

解決的痛點

痛點設計解法
粉絲缺乏與偶像的互動管道個人化會員頁面，查看專屬數據與活動紀錄
周邊商品收藏難以展示展示櫃功能，分系列陳列，鎖頭圖示激勵收集
粉絲間缺乏良性競爭排行榜系統 + 投票機制 + 關注追蹤
粉絲社群變現困難商城整合球場限定周邊、二手拍賣
不同裝置體驗不一致同時設計桌面版與手機版

使用者流程

個人頁面流程

登入 → 我的總覽 → 查看等級/積分/近期活動 → 展示櫃 / 排行榜 / 訂單 / 小遊戲

展示櫃流程

展示櫃 → 瀏覽系列分類 → 已收集：實品圖 / 未收集：鎖頭 → 集滿顯示「CLEAR」

排行榜流程

排行榜 → 月度/季度切換 → 前三名突出展示 → 投票互動 → 獲得更多票數引導
